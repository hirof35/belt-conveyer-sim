# ベルトコンベアシミュレーション (Belt Conveyor Simulation)
<img width="801" height="631" alt="スクリーンショット 2026-06-16 032648" src="https://github.com/user-attachments/assets/3bafc4fa-2af7-4504-93d8-3162f2067808" />

C++ と DXライブラリ（DxLib）を使用して作成した、シンプルな2D物理挙動シミュレーションプログラムです。
ベルトコンベアの上で物体が運ばれ、端に到達すると慣性を保ったまま放物線を描いて自由落下する様子を再現しています。

## 概要

本プログラムは、ゲーム開発や物理シミュレーションの基礎となる「接触判定（足場判定）」と「状態遷移（等速直線運動 ⇄ 放物線運動）」の実装例として作成されました。起動すると、最初からコンベアに乗ったオブジェクトが滑らかに動き出します。

## 主な機能・特徴

- **初期配置の最適化**: 起動時およびリセット時、物体が空中から落下するのではなく、最初からコンベアの上面に完全に接地した状態でスタートします。
- **コンベアスクロール**: 2D描画のトリックを用いて、ベルトコンベアの模様が滑らかにループ移動するアニメーションを実装しています。
- **物理挙動の再現**: 
  - コンベア上：コンベアの移動速度と同調した等速運動（重力無効）
  - コンベア外：離脱した瞬間の水平速度（慣性）を維持したまま、重力加速度によって落下する放物線運動

## 動作環境

- **OS**: Windows 10 / 11
- **開発言語**: C++
- **グラフィックライブラリ**: DXライブラリ (DxLib)
- **開発環境**: Visual Studio 2022 以降推奨

## 操作方法

- **ESC キー**: プログラムを終了します。
- **ループ機能**: オブジェクトが画面外（右または下）に消失すると、自動的にコンベアの初期位置（左端）へリセットされ、シミュレーションが永続的にループします。

## プログラムの構造（主要パラメータ）

ソースコード内の以下の数値を変更することで、シミュレーションの挙動をカスタマイズできます。

| 変数名 | 説明 | 既定値 |
| :--- | :--- | :--- |
| `conveyor.speed` | コンベアの移動速度。大きくすると落下時の飛距離が伸びます。 | `2.5f` |
| `gravity` | 重力加速度。大きくすると物体がより急角度で落下します。 | `0.4f` |
| `box.size` | 運ばれるオブジェクト（箱）の一辺のサイズ。 | `24.0f` |
